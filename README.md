# ATmega lib

## Description

This is a [LibrePCB](https://librepcb.org) library!

- ATmega164P/V/324P/V/644P/V
- ATmega48A/PA/88A/PA/168A/PA/328/P
- ATmega640/1280/2560
- ATmega1281/2561
- AVR ISP

This is my first lib for LibrePCB, let me know if I'm misusing a feature/concept.
